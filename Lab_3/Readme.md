 Реализация блочного шифра «Кузнечик» (Grasshopper)
 согласно ГОСТ Р 34.12-2015.

 Параметры:
   - размер блока: 128 бит (16 байт)
   - размер ключа: 256 бит (32 байта)
   - количество раундов: 10

 Основные преобразования:

	X[k] — побитовый XOR с раундовым ключом
	S    — нелинейная замена байтов (таблица π)
	R    — регистр сдвига с линейной обратной связью (над GF(2^8))
	L    — 16-кратное применение R
